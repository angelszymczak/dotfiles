# Setup Alacritty

```sh
brew install --cask alacrity
```

# Setup Fonts

```sh
brew tap homebrew/cask-fonts
brew install font-meslo-lg-nerd-font
```

# Setup Alacrity Config directory

```sh
mkdir -p ~/.confi/alacritty
touch ~/.config/alacritty/alacrity.toml
```

# Setup Alacritty Theme

```sh
brew install powellevel10k
```
